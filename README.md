# 🛍️ Мерч! - Система учёта товаров

Веб-приложение для управления товарами, поступлениями и продажами магазина одежды.

## ✨ Возможности

- 📊 **Дашборд** с графиками продаж, средним чеком и топ-товарами
- 👕 **Учёт товаров** с категориями, размерами и остатками
- 📦 **Журнал поступлений** с автоматическим пересчётом остатков
- 💰 **Журнал продаж** с отслеживанием продавцов
- 🎯 **Планы продаж** с контролем выполнения
- 👥 **Многопользовательский режим** (владелец + продавцы)
- 📱 **Адаптивный дизайн** (работает с телефона и компьютера)
- 🌓 **Современная тёмная тема**

## 🚀 Быстрый старт

### Шаг 1: Создание Firebase проекта

1. Перейдите на Firebase Console (https://console.firebase.google.com/)
2. Нажмите "Добавить проект"
3. Назовите проект (например, merch-store)
4. Отключите Google Analytics (не нужен)
5. Нажмите "Создать проект"

### Шаг 2: Настройка Firestore Database

1. В левом меню выберите Firestore Database
2. Нажмите "Создать базу данных"
3. Выберите "Запустить в тестовом режиме"
4. Выберите локацию (например, eur3 europe-west)
5. Нажмите "Включить"

### Шаг 3: Настройка Authentication

1. В левом меню выберите Authentication
2. Нажмите "Начать"
3. Перейдите на вкладку Sign-in method
4. Включите Электронная почта/пароль
5. Нажмите "Сохранить"

### Шаг 4: Получение конфигурации Firebase

1. В настройках проекта (⚙️) выберите "Настройки проекта"
2. Прокрутите вниз до раздела "Ваши приложения"
3. Нажмите на иконку Web (</>)
4. Зарегистрируйте приложение (название любое)
5. Скопируйте конфигурацию firebaseConfig

### Шаг 5: Вставка конфигурации в код

1. Откройте файл index.html
2. Найдите блок с firebaseConfig (строки 180-190)
3. Замените значения на свои (YOUR_API_KEY, YOUR_PROJECT и т.д.)

### Шаг 6: Деплой на GitHub Pages

1. Создайте репозиторий на GitHub (например, merch-store)
2. Загрузите все файлы в репозиторий через git
3. В настройках репозитория Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main, folder: / (root)
4. Через 1-2 минуты сайт будет доступен

## 👥 Регистрация пользователей

### Первый пользователь (Владелец)

1. Откройте сайт
2. Перейдите на вкладку "Регистрация"
3. Заполните данные и выберите роль "Владелец"
4. Нажмите "Зарегистрироваться"

### Дополнительные пользователи (Продавцы)

1. Зарегистрируйте продавца с ролью "Продавец"
2. Продавцы видят только базовые разделы (без планов)

## 📱 Использование на телефоне

Приложение полностью адаптивно. Добавьте на главный экран для быстрого доступа.

## 🎯 Основные операции

### Добавление товара
Товары → + Добавить товар → Заполнить форму → Сохранить

### Приёмка товара
Поступления → + Добавить поступление → Выбрать товар и количество → Сохранить

### Регистрация продажи
Продажи → + Добавить продажу → Выбрать товар и количество → Сохранить

### Установка плана
Планы → + Добавить план → Указать период и сумму → Сохранить

## 📞 Поддержка

При проблемах проверьте:
- Консоль браузера (F12) на наличие ошибок
- Правильность Firebase конфигурации
- Включены ли Firestore и Authentication
- Настроены ли правила безопасности
